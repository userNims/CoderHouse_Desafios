DESAFIO 1

Consigna: Tomando como base los ejemplos anteriores de la estructura for y while, crear un algoritmo que repita un bloque de instrucciones. En cada repetición es necesario efectuar una operación o comparación para obtener una salida por alerta o consola.

Solución - Hablar con las Ballenas

Vamos a traducir palabras o frases humanas a su equivalente en lenguaje ballena. Para ello se siguen algunas reglas:

  1.- El lenguaje de ballena, que se traduce, solo tendrá vocales.
  2.- Las vocales como 'u' y 'e' se duplicarán.

Luego, se mostrara mediante una alerta el texto traducido.
La misma debera pronunciarse lentamente y en voz alta.

  Ej: Tortuga (lenguaje humano) -> OUUA (lenguaje de ballena)

Para el ejercicio se hizo uso de bucles FOR anidados y de un bucle WHILE.
