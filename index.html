DESAFIO 1 CICLOS/ITERACIONES

Consigna: Tomando como base los ejemplos anteriores de la estructura for y while, crear un algoritmo que repita un bloque de instrucciones. En cada repetición es necesario efectuar una operación o comparación para obtener una salida por alerta o consola.

Solución - Hablar con las ballenas

Vamos a trasladar palabras o frases humanas a su equivalente en lenguaje ballena. Para ello se siguen algunas reglas:

  1.- El lenguaje de ballena, que se traduce, solo tiene vocales.
  2.- Las vocales como 'u' y 'e' serán duplicadas.

Luego, se mostrara mediante una alerta el texto traducido.
La misma debera pronunciarse en voz alta y de forma un poco lenta.

  Ej: Tortuga (lenguaje humano) -> OUUA (lenguaje de ballena)

Para el ejercicio se hizo uso de bucles FOR anidados y de un bucle WHILE.

DESAFIO 2 - FUNCIONES

>> Consigna: Armar la estructura HTML del proyecto incorporando lo ejercitado en las clases anteriores, algoritmo condicional y algoritmo con ciclo. Utilizar funciones para realizar esas operaciones.

Solución - Piedra, Papel o Tijera

Recrearemos el clasico juego para dos personas, 'piedra papel tijera'. Para ello se pide al usuario que ingrese una opción valida y se la compara con una opción de la computadora.

Las posibles respuestas son:
  -Piedra destruye tijeras.
  -Tijeras cortan papel.
  -Papel envuelve piedra.
  -Tambien puede existir la opción de un empate.

Nuestro código se divide principalmente en cuatro partes:
  -Conseguir y validar la opción del usuario.
  -Conseguir de forma aleatoria la opción de la computadora.
  -Comparar las dos opciones y determinar quién es el ganador.
  -Mostrar los resultados de la partida.

Solución - Crear una historia

Completaremos una historia divertida con palabras, verbos, frases, animales y el nombre de una persona. Estos datos los iremos guardando en un array para luego usarlos.

Nuestro código se divide principalmente en cuatro partes:
  -Conseguir y validar los datos que ingresa el usuario.
  -Almacenar dichos datos en un array.
  -Mostrar los datos que se ingresaron haciendo uso de los métodos de array concat() y join().
  -Mostrar la historia completa, para lo cual llamamos a los elementos del array que creamos previamente.
